# 🧠 BrainBattle

> Командный квиз в реальном времени | QuizBattle от Сбера

**BrainBattle** — веб-приложение для проведения командных квиз-игр в реальном времени. Два участника соревнуются, отвечая на вопросы по выбранной теме. Вопросы генерирует AI (GPT-4o), есть резервные вопросы по всем темам.

---

## Возможности

- **AI-генерация вопросов** — GPT-4o автоматически создаёт вопросы по любой теме
- **Командный режим** — 2 команды (А и Б) соревнуются поочерёдно  
- **Режим "Каждый за себя"** — все игроки против всех
- **WebSocket в реальном времени** — мгновенная синхронизация состояния
- **Выбор сложности** — лёгкий / средний / сложный
- **Бонус за скорость** — быстрее ответил = больше очков
- **Таймер** — настраиваемое время на вопрос (15–60 сек)
- **Пауза** — ведущий может остановить игру
- **Пропуск вопроса** — ведущий может пропустить
- **Дисквалификация** — ведущий удаляет игрока
- **Статистика** — подробная аналитика после игры
- **Экспорт результатов** — скачать в JSON
- **Рейтинг** — таблица лидеров по всем играм
- **Регистрация и профили** — (необязательна для игры)
- **Защита паролем** — приватные комнаты
- **Возврат в игру** — кнопка возврата при перезагрузке
- **Мобильная адаптация** — работает на телефоне и планшете
- **Звуковые эффекты** — правильный/неправильный ответ, тик таймера
- **Очередь внутри команды** — в командном режиме каждый отвечает по очереди
- **Резервные вопросы** — если AI недоступен, используются предзаготовленные

## Стек технологий

### Backend
- **Python 3.11** + **FastAPI**
- **WebSocket** (нативный FastAPI)
- **MongoDB** (Motor async driver)
- **JWT** авторизация (python-jose)
- **bcrypt** для хеширования паролей
- **emergentintegrations** для OpenAI GPT-4o

### Frontend
- **React 19** + React Router
- **Framer Motion** — анимации
- **Tailwind CSS** — стилизация
- **Space Mono** + **Syne** — типографика (Google Fonts)

## Установка и запуск

### Требования
- Python 3.9+
- Node.js 18+
- MongoDB
- yarn

### Backend
```bash
cd backend
pip install -r requirements.txt
# Настройте .env файл (см. ниже)
uvicorn server:app --host 0.0.0.0 --port 8001 --reload
```

### Frontend
```bash
cd frontend
yarn install
yarn start
```

### .env Backend
```env
MONGO_URL=mongodb://localhost:27017
DB_NAME=brainbattle_db
EMERGENT_LLM_KEY=ваш_ключ
JWT_SECRET=ваш_секрет
CORS_ORIGINS=*
```

### .env Frontend
```env
REACT_APP_BACKEND_URL=http://localhost:8001
```

## Как играть

1. **Ведущий** переходит на сайт → «СОЗДАТЬ ИГРУ»
2. Вводит тему, выбирает количество вопросов и сложность
3. Нажимает «Создать игру» — AI генерирует вопросы
4. Получает **6-значный PIN код** и отправляет друзьям
5. **Игроки** переходят на сайт → «ВОЙТИ В ИГРУ», вводят PIN и имя
6. Можно **выбрать команду** в лобби
7. Ведущий нажимает **«НАЧАТЬ ИГРУ»**
8. Командам задаются вопросы поочерёдно
9. После всех вопросов — итоговые результаты и статистика

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| POST | /api/games/create | Создать игру (AI генерация вопросов) |
| POST | /api/games/join | Присоединиться к игре |
| GET  | /api/games/{pin} | Получить состояние игры |
| POST | /api/games/{id}/start | Начать игру (ведущий) |
| POST | /api/games/{id}/action | Игровые действия (ответ, пауза, пропуск) |
| POST | /api/games/choose-team | Выбрать команду |
| GET  | /api/leaderboard | Таблица лидеров |
| POST | /api/auth/register | Регистрация |
| POST | /api/auth/login | Авторизация |
| WS   | /ws/{game_id}/{player_id} | WebSocket соединение |
